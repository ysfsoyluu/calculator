# Simple Calculator 🧮

A beginner Python project that performs basic math operations through a clean command-line interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division (with zero division protection)
- Input validation — handles invalid entries gracefully
- Loop menu — keeps running until you choose to quit

## How to Run

Make sure you have Python installed, then run:

```
python calculator.py
```

## Example

```
==========================
   Simple Calculator 🧮   
==========================
Operations:
  1 — Addition       (+)
  2 — Subtraction    (-)
  3 — Multiplication (x)
  4 — Division       (/)
  5 — Quit
--------------------------

Choose an operation (1/2/3/4/5): 1
Enter first number: 15
Enter second number: 7
Result: 15.0 + 7.0 = 22.0
```
